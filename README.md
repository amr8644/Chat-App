## Tech Stack

**Client:** Next JS and Material UI

**Server:** Golang and MySQL

## Run Locally

Clone the project

```bash
  git clone https://github.com/amr8644/Chat-App.git
```

Go to the project directory

```bash
  cd Chat-App
```

## For Client Side

```bash
  cd client
```

Install dependencies

```bash
  npm install
```

Run

```bash
  npm run dev
```

## For Server Side

```bash
  cd server/cmd
```

Build project

```bash
  go build main.go -o main
```

Start server

```bash
./main
```
